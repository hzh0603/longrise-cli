## 安装
	npm i longrise-cli -g

## 相关命令
> 查看全部命令 longrise

> 查看可使用模板 longrise ls

> 查看所有模板的git地址 longrise ls_git

> 新增模板 longrise add

> 删除模板 longrise delete <name>

> 使用么个模板 longrise use <template> (longrise use React-BBW)
